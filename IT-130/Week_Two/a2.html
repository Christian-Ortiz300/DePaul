Q1.What is an HTML "comment"? why is commenting important when writing code?

an HTML comment is a tag where you add comments in regards to your HTML source. It looks like this: < !---- >. HTML comments helps with communicating with either yourself and with others. It can often remind someone to do something or can help communicate with another person who is looking at their code.

Q2.Many tags and attributes have been "deprecated" over the years. Included among these are the <font> tag (for changing the font), the <u> tag (for underlying), and the <center> tag for centering text. Why have these tags been deprecated?

The reason is because one can use CSS to "style" it. therefore theres no need of it to be changed within the HTML file when it can be done in CSS. Therefore, it has led them to be phased out of the HTML standard.
